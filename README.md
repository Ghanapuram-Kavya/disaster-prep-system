# Disaster Preparedness and Response Education System

## 📌 Overview
This project is a digital platform designed to improve disaster awareness and emergency preparedness among students and institutions.

## 🚀 Features
- Interactive learning modules
- Quiz and assessment system
- Real-time alerts
- SOS emergency system
- Progress tracking with badges and certificates

## 🛠 Technologies Used
- Frontend: React.js
- Backend: Flask
- Database: MySQL

## 🔐 Note
Sensitive data such as API keys and database credentials are stored securely using environment variables (.env) and are not included in this repository.

## ▶️ How to Run
1. Clone the repository
2. Install dependencies
3. Run backend and frontend servers
